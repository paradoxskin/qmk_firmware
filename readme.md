# CIDOO QK61 - QMK Firmware Fork

A customizable 85-key keyboard with FS026 chipset.

## About this fork

This is a fork of [qmk/qmk_firmware](https://github.com/qmk/qmk_firmware) with **custom FS026 microcontroller support** and proprietary drivers for CIDOO QK61.

### Branches

- **master**: Synchronized with upstream QMK (without custom changes)
- **qk61**: FS026 chipset support + CIDOO QK61 firmware (use this for building)
